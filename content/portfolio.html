Gotta Go

Gotta Go is a React-based web application designed to crowdsource restrooms for frequent drivers. It features MongoDB, a NodeJS server, and Bulma styling. Currently, the Gotta Go development team is fine-tuning user authentication, maps, and Redux.

Friend Finder

This application uses Node.js and JavaScript to calculate user compatibility with character profiles from "Stranger Things." The application loops through each user's survey answers in comparison to profiles in order to find the friends with most similar answers.
